To run this using IntelliJ IDEA you will need to download Tomcat 9.0.34 and configure it by add
configuration or edit configuration option. 

1. You can either use apache-tomcat-9.0.34 directory of the repository as Tomcat home or you can
download and extract the binary.

We have also checked in the apache-tomcat-9.0.34 server which we used for our development purposes. Under the ./apache-tomcat-9.0.34/lib directory we have added all the dependency jars which is required for the overall system to run.
